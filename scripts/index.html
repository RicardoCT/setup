Scripts

Esta carpeta tiene los siguientes scripts:

autorizacion.php Modelo simple de autenticación / autorización.
buscador.php Crear un formulario de búsqueda y página para visualizar los resultados de búsqueda
contacto.php Genera un formulario de contacto que envía los datos a una dirección de correo electrónico.
representar.php Representa los datos de una tabla y permite hacer las operaciones básicas con los mismos (Crear, Leer, Actualizar, Eliminar).
union.php Vincula muchos elementos de una tabla con muchos elementos de otra tabla.
